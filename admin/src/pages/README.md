# Admin Dashboard Pages

HTML pages for the Admin Dashboard application.

## Pages Overview

### Core Pages

- **dashboard.html** - Main admin dashboard with analytics
- **accounts.html** - User account management
- **manage-profile.html** - Admin profile management
- **change-password.html** - Password change functionality

### Academic Management

- **courses.html** - Course management
- **competencies.html** - Competencies framework
- **schedule.html** - Schedule management
- **graduates.html** - Graduate tracking

### Student Management

- **admission.html** - Admission processing
- **application.html** - Application management
- **enrollment.html** - Enrollment processing
- **attendance.html** - Attendance records
- **requests.html** - Student requests

### Inventory Management

- **audit-inventory.html** - Audit inventory tracking
- **caate-inventory.html** - CAATE inventory management

## Page Structure

Each page typically includes:

- Header with navigation
- Sidebar menu
- Main content area
- Page-specific functionality
- Footer

## Development

When creating new pages:

1. Use consistent layout structure
2. Include corresponding CSS file in `../assets/css/`
3. Include corresponding JS file in `../assets/js/`
4. Follow naming conventions (kebab-case)
5. Ensure responsive design
6. Add accessibility features
