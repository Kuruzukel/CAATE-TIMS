# Landing Page Pages

HTML pages for the Landing Page Website.

## Pages Overview

### Main Pages

- **index.html** - Landing page with hero section and featured courses

### Course Pages (courses/)

1. **skincare.html** - Skincare course information
2. **nailcare.html** - Nail care course information
3. **advanced-skincare.html** - Advanced skincare course
4. **aesthetic.html** - Aesthetic services course
5. **collagen-hairloss.html** - Collagen & hair loss treatment course
6. **eyelash-eyebrow.html** - Eyelash & eyebrow course
7. **facial-peeling.html** - Facial peeling course
8. **light-heat-therapy.html** - Light & heat therapy course
9. **permanent-makeup.html** - Permanent makeup course
10. **trainers.html** - Trainers directory

## Page Structure

Each course page typically includes:

- Course title and hero image
- Course overview
- Curriculum/modules
- Duration and schedule
- Trainer information
- Enrollment call-to-action
- Contact information

## Development

When creating new course pages:

1. Use consistent layout structure
2. Include course-specific CSS
3. Include course-specific JavaScript
4. Optimize images
5. Add proper meta tags for SEO
6. Ensure responsive design
7. Test on multiple devices

## SEO Considerations

- Use descriptive page titles
- Include meta descriptions
- Add Open Graph tags
- Use semantic HTML
- Optimize images with alt text
